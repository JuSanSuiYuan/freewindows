# Material3Demo

Material 3 风格的 Qt 演示项目

## 构建

### 使用 CMake

```powershell
# 配置
cmake -B build -G Ninja

# 构建
cmake --build build

# 运行
.\build\Material3Demo.exe
```

### 使用 Qt Creator

1. 打开 Qt Creator
2. 文件 → 打开文件或项目
3. 选择 CMakeLists.txt
4. 点击构建并运行

## 特性

- ✅ 圆角窗口
- ✅ Material 3 渐变背景
- ✅ 带阴影的按钮
- ✅ 流畅动画
- ✅ 可拖动标题栏
- ✅ 现代化外观

## 技术栈

- Qt 5/6
- CMake
- C++17
