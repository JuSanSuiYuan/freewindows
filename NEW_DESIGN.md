### 布局

```
╭─────────────────────────────────────────────────╮
│ × − □ │    ◯ Material 3 Demo    │ − □ × │

  ↑                ↑                    ↑
```

### 左侧按钮（macOS 风格）

- **×** - 关闭（红色悬停）
- **−** - 最小化（黄色悬停）
- **□** - 最大化/还原（绿色悬停）

### 中间

- **◯** - 应用图标
- **Material 3 Demo** - 窗口标题

### 右侧按钮（Windows 风格）

- **−** - 最小化（灰色悬停）
- **□** - 最大化/还原（灰色悬停）
- **×** - 关闭（红色悬停）

## 颜色代码

### 背景渐变

```cpp
gradient.setColorAt(0, QColor("#87CEEB"));    // 天蓝色
gradient.setColorAt(0.5, QColor("#7CFC00"));  // 草绿色
gradient.setColorAt(1, QColor("#FF69B4"));    // 粉红色
```

### 按钮颜色

```cpp
"#87CEEB"  // 天蓝色 (Sky Blue)
"#7CFC00"  // 草绿色 (Lawn Green)
"#FF69B4"  // 粉红色 (Hot Pink)
```

### 悬停颜色

```cpp
QColor(255, 0, 0)    // 红色 - 关闭按钮
QColor(255, 255, 0)  // 黄色 - 左侧最小化
QColor(0, 255, 0)    // 绿色 - 左侧最大化
QColor(128, 128, 128) // 灰色 - 右侧按钮
```

## 自定义颜色

### 修改背景渐变

```cpp
// 改成你喜欢的颜色
gradient.setColorAt(0, QColor("#你的颜色1"));
gradient.setColorAt(0.5, QColor("#你的颜色2"));
gradient.setColorAt(1, QColor("#你的颜色3"));
```

编辑第 70 行：

```cpp
contentLayout->addWidget(createMaterial3Button("按钮1", "#你的颜色"));
```
